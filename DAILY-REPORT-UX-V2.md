# تحسين شاشة التقرير المعتمد

- تكبير شارة اعتماد التقرير وتبسيط نصها.
- حذف رقم التقرير القديم والرسائل التقنية.
- إضافة وحدات واضحة لعدد المواقع والصور.
- تحويل موعد التقرير القادم إلى بطاقة مختصرة: غدًا، 00:00 بتوقيت مكة.
- إزالة زر الطباعة من صفحة المقاول.
- إبراز زر العودة للمشاريع كالإجراء الوحيد بعد الاعتماد.
- تصغير سطر آخر مزامنة ووضعه في أسفل البطاقة.
