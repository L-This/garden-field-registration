# Garden Field Registration

مستودع مستقل لتسجيل ري الحدائق ميدانيًا.

## المحتوى

- صفحة رئيسية تعرض المشاريع الأربعة.
- صفحة تسجيل لكل مشروع.
- بطاقات حدائق مع رفع صورة وجلب موقع.
- حفظ مؤقت داخل المتصفح.
- إرسال جماعي للتقرير.
- تصميم عربي RTL مناسب للجوال.

## التشغيل المحلي

```bash
npm install
npm run dev
```

## الربط مع الخلفية

أضف متغير البيئة التالي في Vercel:

```bash
NEXT_PUBLIC_FIELD_API_URL=https://YOUR_BACKEND_ENDPOINT
```

إذا لم يتم تعريف الرابط، يعمل النظام بوضع محاكاة للإرسال.

## أين أضيف الحدائق؟

عدّل الملف:

```text
src/data/gardens.ts
```

## أين أضيف المشاريع؟

عدّل الملف:

```text
src/data/projects.ts
```

## ملاحظات إنتاجية

- لا تضع مفاتيح سرية داخل هذا المستودع.
- اجعل صلاحيات الإدارة في مستودع لوحة الإدارة فقط.
- استخدم Backend منفصل لمعالجة الصور ومنع التكرار الحقيقي.
